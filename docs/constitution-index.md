# Constitution Index

## Purpose

This file is the registry for the ASCOS Constitution framework. It provides the chapter inventory and lifecycle status for all Constitution content.

## Registry model

The Constitution is managed through a chapter registry. Each entry below captures the metadata required by the framework: identifier, title, lifecycle status, owner, version, review cadence, and dependent standards.
